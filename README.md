# task-board-tkinter


task-board with python.


## ⁉ O que é o Task Board?

- O Task Board é uma ferramenta de organização.


## 🖥 Requisitos

- Ter o [Python 3+](http://www.python.org/) instalado na máquina.

## 🎡 Como Rodar

- Entrar na pasta do arquivo main.py e rodar o comando 
```
python main.py
```
